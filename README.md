# 物资管理小程序前端

## Content
Uniapp项目，使用vant组件。

## 启动方法
- 安装依赖
    ```sh
    npm install
    ```
- 测试
    ```sh
    npm run dev:mp-wexin
    ```
    同时需要开启微信开发者工具，选择打开项目，之后选择`./dist/dev/mp-weixin` 作为项目文件夹。之后编译运行即可。

- 打包
    ```sh
    npm run build:mp-weixin
    ```
